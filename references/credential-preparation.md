# Credential Preparation

Study and practice frameworks for coaching credentials and judging courses in the
CrossFit ecosystem. Used by `crossfit-credential-preparation-coach` and
`crossfit-coach-performance-reviewer`.

## Integrity rules (non-negotiable)

- **Generate original practice material only.** Never reproduce, paraphrase from
  memory, or reconstruct confidential or copyrighted exam questions, answer keys,
  or proprietary course manuals.
- **No guarantees.** Studying improves readiness; it does not guarantee passing.
  Say so.
- Build study plans from **publicly described** topic areas and from first
  principles of coaching, not from leaked content.

## Credential modes (distinct study targets)

| Mode | Focus of preparation |
| --- | --- |
| Level 1 (foundational) | Core movements & standards, methodology basics, scaling, safety, basic coaching — broad recall + understanding. |
| Level 2 (applied coaching) | Live coaching practice: seeing/correcting faults, cueing, lesson delivery, class management, scaling decisions. |
| Level 3 (trainer / written) | Deeper methodology, programming, application; written-exam-style reasoning. |
| Level 4 (evaluation) | Demonstrated coaching competence under evaluation; high-bar application and consistency. |
| Judges Course | Movement standards, no-rep criteria, judging mechanics, consistency. |
| Advanced judging | Edge cases, high-pressure consistency, video/standards adjudication. |
| Spot the Flaw | Rapid fault identification from demonstration. |

## Study toolkit (build these, original content only)

- **Topic maps** — break the mode into domains; rate confidence per domain.
- **Study plan** — schedule by gap size and exam date; spaced repetition.
- **Flashcards** — original Q/A on standards, terminology, methodology.
- **Scenario questions** — "an athlete does X, what do you do?" coaching
  judgment items.
- **Mock exams** — original, mode-appropriate format and difficulty.
- **Fault-ID drills** — describe a movement error; practice naming the fault,
  the correction, and the cue (ties to the fault analyzer and cue generator).
- **Practical practice** — for applied modes: lesson-planning reps, group-
  management drills, demo practice, timed fault-spotting.
- **Gap analysis & remediation** — after each mock, map misses to topics and
  prescribe targeted review.

## Coaching-performance review (for L2+/practical readiness)

Evaluate against the observable coaching skills: lesson plan, explanation, demo,
observation, fault identification, correction, cue selection, presence, group
management, scaling, safety, time management, engagement. Output evidence-based
feedback, the single highest-priority development area, a practical drill, and a
follow-up rubric. See the `crossfit-coach-performance-reviewer` skill.
