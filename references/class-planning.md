# Class Planning

How to convert a workout into a coached group-class experience and run the floor.
Used by `crossfit-class-plan-builder`, `crossfit-class-logistics-and-floor-manager`,
and `crossfit-foundations-and-onramp-builder`. Default class length is 60 minutes;
support other lengths by scaling each block proportionally.

## The 60-minute template (adjust block lengths to fit other durations)

| Block | Approx time | Purpose |
| --- | --- | --- |
| Pre-class prep | before | equipment counts laid out, whiteboard written |
| Arrival / intro | 0–2 | greet, scan for injuries/absences, frame the day |
| Whiteboard brief | 2–5 | stimulus intent, time domain, target score, scaling map |
| General warm-up | 5–12 | raise temperature, monostructural + mobility |
| Specific warm-up | 12–22 | movement-prep for the day's exact patterns |
| Teaching / skill | 22–32 | teach/refine the technical movement(s) |
| Rehearsal / build | 32–42 | practice at workout pace; build to working loads |
| Setup & brief | 42–46 | stations, standards, scaling choices confirmed |
| Workout | 46–56 | the training piece; coach the floor |
| Recovery / cooldown | 56–59 | bring down, brief mobility |
| Cleanup / notes | 59–60 | reset equipment, log notes |

Build **buffers** in and have a **late-running contingency** (what to cut if the
class is behind — usually trim the build or shorten cooldown, never the safety
brief or warm-up of a heavy/high-skill piece).

## What a class plan must specify

Exact timing per block, equipment setup, coach positioning, the 2–3 teaching
points per movement, scaling checkpoints (when the coach confirms each athlete's
choice), transition logic, and the contingency. See
[workout-design-framework.md](workout-design-framework.md) for the underlying
workout.

## Floor management & logistics

When attendance, equipment, or space is constrained:

- **Equipment math:** count athletes vs. barbells/racks/ergs/rig stations; if
  short, use **heats**, **station rotations**, or **partner/relay** formats.
- **Heats:** assign by ability or equipment; stagger starts; define who counts/
  judges. Provide explicit heat assignments and timing.
- **Station maps:** lay out a text map of the floor (stations, rotation
  direction, safety zones, traffic flow) so setup is unambiguous.
- **Coach visibility:** position so the whole floor is in view, especially during
  overhead/loaded work; name safety zones (barbell drop areas, running lanes).
- **Mixed ability:** scale within the same stimulus so the class stays together;
  use the scaling map from the brief.
- **Setup/cleanup:** assign and time it; reset for the next class.

## Onboarding / on-ramp

Foundations programs teach the patterns, key lifts, gymnastics basics,
monostructural work, formats, intensity, scaling, safety, etiquette, terminology,
and scoring before an athlete enters regular class. Support 1:1, small-group,
accelerated, and multiweek formats; gate progression on demonstrated competence,
not attendance count.
