# Athlete Intake

The fields coaching skills draw on to individualize output. **Not every field is
required for every request.** Each skill names its own required vs. optional
inputs. The rule across all skills: never fabricate athlete data; when a
materially decision-changing field is missing, either ask for it or return a
clearly labeled provisional answer with the assumption stated.

## Identity & context

- **Training age** — months/years of consistent structured training (distinct
  from chronological age).
- **Chronological age / life stage** — relevant for masters, youth, pregnancy/
  postpartum (see [safety-and-referral.md](safety-and-referral.md)).
- **Primary goal & timeline** — GPP, a competition on a date, a specific
  movement/lift, body composition, return from layoff.
- **Schedule** — sessions/week, session length, days available, other sport/work
  load.
- **Equipment access** — full affiliate, home/garage, travel/minimal; specific
  constraints (one barbell, no rig, limited ergs).

## Capacity snapshot (use what exists; do not require all)

- **Strength** — key 1RM/3RM or working loads: back squat, front squat,
  deadlift, press, bench.
- **Olympic lifting** — snatch, clean & jerk maxes and technical proficiency.
- **Gymnastics** — pulling (pull-ups, C2B, muscle-ups), pressing (HSPU,
  handstand hold/walk), core, and skill ceilings; what's strict vs. kipping.
- **Monostructural** — benchmark times/paces (e.g. 2k row, 1-mile run, max
  cals in X min) and pacing discipline.
- **Mixed-modal benchmarks** — named workout scores with date and any scaling
  used.

## Constraints & history

- **Injury / pain history** — current pain, past injuries, clinician
  restrictions, movements that flare. Pain that limits movement triggers the
  safety framework before programming.
- **Medical conditions / pregnancy-postpartum status** — defer to qualified
  professionals where indicated; coach only within clearance.
- **Recovery inputs** — sleep, stress, nutrition adequacy, life load.

## Behavior & adherence

- **Consistency** — attendance/adherence patterns.
- **Scaling history** — what the athlete typically scales and why.
- **Preferences / dislikes** — movements they avoid (and whether avoidance maps
  to a real limiter — a weakness-prioritization question, not just a preference).

## Minimum viable intake by request type

| Request | Usually need at minimum | Safe to assume / ask later |
| --- | --- | --- |
| Single workout | level, equipment, time, any injuries | exact maxes |
| Multiweek cycle | goal, timeline, sessions/week, level, equipment, injuries | fine-grained benchmarks |
| Scaling | the workout + intended stimulus, athlete level, injuries | full capacity profile |
| Assessment | available test results + goals | complete history |
| Competition prep | event/season/division + dates + athlete level | every 1RM |

When required fields are missing, provide a provisional answer using conservative
defaults, state each assumption, and ask only for the fields that would
materially change the prescription.
