# Example 3 — Mixed-ability class, one shared stimulus (incomplete info)

**Prompt:** "Big mixed class — some RX athletes, some on week two. How do I run the
floor so everyone trains the same stimulus and I don't lose half of them?"

**Context:** Athlete count not given; equipment not given. Workout is a single
mono-structural + gymnastics couplet with a scaling map already on the whiteboard.

---

## Note on missing inputs (provisional)
**Assumption:** ~20 athletes and roughly 1 erg/rig station per athlete (typical
medium box). The math and map below are **provisional** — re-run them with the real
counts. The one or two numbers that would change the format: **how many athletes**
and **how many erg/rig stations**.

## Attendance & equipment inventory (provisional)
- Athletes: **~20 (assumed)** · Stations: **~1 per athlete (assumed)**
- Constraint here is **ability spread**, not gear — so the design goal is keeping
  one shared stimulus, not solving a shortage.

## Chosen format — single floor, one stimulus, scaled in place
With ~1 station per athlete, everyone works at once on a single clock. Keep the
class together by **scaling within the same stimulus** (same time domain, same
"hard but repeatable" intent) using the whiteboard scaling map — RX, intermediate,
and beginner all finish in the same window. See
[../../../references/coaching-principles.md](../../../references/coaching-principles.md)
§3 (stimulus) and §2 (mechanics → consistency → intensity).

## Text station map
```
  [ RIG WALL: pull-up / ring stations 1..n ]
  -------------------------------------------
  |            SAFETY / WALK BUFFER          |
  -------------------------------------------
  [ ERG ROW: rowers / bikes 1..n, facing rig ]
```
Pair each athlete to one rig spot and one erg; they shuttle the short buffer
between them. Slower-scaled athletes take the rig spots nearest the coach for
closer eyes.

## Heat assignments & timing
Single heat, single clock. Group the newest athletes closest to the coach. No
staggering needed at ~1:1 stations.

## Transition logic
Athletes own a rig spot and an erg and move only across the buffer between them —
no cross-floor traffic. The scaled-down version uses the same stations, just fewer
reps / assisted variation, so transitions look identical for everyone.

## Setup & cleanup plan
- **Pre-class:** bands/boxes for assisted gymnastics staged at the rig; ergs zeroed.
- **Cleanup:** bands and boxes back to the rig wall; ergs reset.

## Coaching/visibility notes
Stand at the erg end facing the rig so every gymnastics station is visible. With
mixed ability the watch priority is **form breakdown on the scaled gymnastics
movement under fatigue** — call earlier breaks before standards slip.
