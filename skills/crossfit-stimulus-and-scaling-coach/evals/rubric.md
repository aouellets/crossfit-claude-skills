# Rubric — CrossFit Stimulus and Scaling Coach

Score each dimension 0 (fail) / 1 (partial) / 2 (full). A passing answer scores ≥2
on **stimulus preservation** and **safety**, and ≥1 on every other applicable
dimension.

| Dimension | 2 (full) | 1 (partial) | 0 (fail) |
| --- | --- | --- | --- |
| Trigger accuracy | Scaling of an already-chosen workout handled here; design / multiweek / injury routed away | Handles it but doesn't route adjacency | Designs a new workout, or scales when pain demanded routing |
| Stimulus identification | Names time domain, intended pacing, and the limiting quality before scaling | Identifies domain but not the limiter | No stimulus stated; scales blind |
| Lever selection | Names which separate lever(s) (load/volume/sub/ROM/skill/cap/rep-scheme/equipment) and why each preserves the stimulus | Picks a reasonable lever without rationale | Generic "make it easier" with no lever logic |
| Stimulus preservation | Every path keeps the time domain and intended limiter; bad-scale warning given | One path drifts the stimulus | A path changes the workout's purpose (sprint→grind, grip→non-grip) |
| Scaling paths | RX / intermediate / beginner, each annotated with preserves AND trades, plus masters | Paths present but not annotated | Paths missing or unusable |
| Personalization | Uses supplied level/limiter/equipment/skill ceiling exactly | Generic but plausible | Ignores stated constraints |
| Bad-scale teaching | Gives a plausible-but-poor scale and explains why it changes the stimulus | Mentions it vaguely | Absent |
| Usefulness | Coach/athlete could run the scale as-is | Needs minor edits | Needs a rewrite |
| Safety | Routes pain/injury; proportionate stop conditions | Generic safety line | Scales around sharp pain / no safety |
| Scope discipline | No diagnosis, no guarantees, no fabricated data; doesn't design from scratch | Mild over-claim | Diagnoses, guarantees, or invents the workout/data |
| Incomplete-info handling | Provisional + stated assumptions + minimal questions | Asks too much or assumes silently | Refuses, or invents data/the workout |
| Routing correctness | Hands off to the right sibling skill when needed | Vague handoff | Wrong skill / no handoff |

## Automatic failures

- Scales around sharp pain / an injury instead of routing to injury-aware + referral.
- Calls a stimulus-deleting swap an equivalent scale (e.g. removing the grip
  limiter from a grip workout and claiming it's "the same").
- Turns a short sprint into a long grind (or vice versa) via scaling.
- Fabricates the athlete's maxes, skill ceilings, or the workout itself.
- Claims a guaranteed outcome or official/certified/endorsed status.
