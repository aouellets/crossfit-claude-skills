# Example 3 — 60-minute plan with a high-skill teaching emphasis

**Prompt:** "Build a class plan — the workout has muscle-ups and the class is mixed,
so I want the teaching to carry the hour."

**Context:** Workout given —
**For time (15-min cap):** 3 rounds of 5 ring muscle-ups + 10 dumbbell snatches
(50/35 lb · 22.5/15 kg) + 15 box jump-overs. Mixed ability, ~12 athletes. The
muscle-up is the limiter, so teaching/rehearsal blocks expand within the 60-min
template ([../../../references/class-planning.md](../../../references/class-planning.md)).

---

## Minute-by-minute timeline
| Block | Clock | Coach action |
| --- | --- | --- |
| Pre-class prep | before | Rings set to height, DBs/boxes counted, whiteboard written |
| Arrival / intro | 0–2 | Greet, scan for shoulder niggles, frame the muscle-up focus |
| Whiteboard brief | 2–5 | Stimulus, cap, and the muscle-up scaling ladder up front |
| General warm-up | 5–11 | Monostructural + shoulder/scap prep, hollow/arch drills |
| Specific warm-up | 11–20 | Ring rows, transition drills, DB snatch + box step prep |
| Teaching / skill | 20–36 | **Expanded:** muscle-up — false grip, hips to rings, fast turnover, sternum-to-rings standard; assign each athlete a tier |
| Rehearsal / build | 36–44 | Practice the assigned muscle-up tier + 3 DB snatches + 3 box jump-overs at pace |
| Setup & brief | 44–47 | Lock each athlete's muscle-up scaling; restate cap and standards |
| Workout | 47–55 | Run the floor; tight watch on rings under fatigue |
| Recovery / cooldown | 55–58 | Shoulder/lat down-regulation |
| Cleanup / notes | 58–60 | Reset rings/boxes, log scaling tier used |

## Per-movement teaching points
- **Ring muscle-up (high skill — the hour's emphasis):** (1) false grip + hollow
  setup; (2) hips drive to the rings, fast turnover through the dip; (3) standard =
  sternum to rings + full support lockout. Scaling ladder: jumping MU → low-ring
  transitions → banded → chest-to-ring pull + ring dip.
- **DB snatch:** (1) hinge and drive, do not muscle it; (2) lock out fully overhead.
- **Box jump-over:** (1) soft landing; (2) step down option for high reps/fatigue.

## Scaling checkpoints
1. Specific warm-up (can the athlete hold a false grip / show a transition?).
2. Teaching block (assign each athlete a muscle-up tier explicitly).
3. Rehearsal (confirm the tier survives at pace; regress if not).
4. Setup brief (lock it on the board so no one self-promotes mid-workout).

## Coach positioning
Teaching/rehearsal: directly under the rings to spot transitions. Workout: rig-side
with the whole floor visible; prioritize eyes on every athlete attempting muscle-ups
as they fatigue — that is where form fails.

## Transitions
Athletes stay at their ring/DB/box station; mixed tiers run side by side without
reshuffling the floor.

## Buffers + late-running contingency
~3 min slack across warm-up/build. **If behind:** trim the box-jump prep and shorten
cooldown. Do **not** cut the muscle-up teaching or warm-up — that is the whole point
of the hour and the highest-risk movement.

## Coach notes
For athletes not yet near a muscle-up, the assigned tier is the win for the day — do
not let "RX or nothing" creep in. Note who is one tier away so the next plan
progresses them.
