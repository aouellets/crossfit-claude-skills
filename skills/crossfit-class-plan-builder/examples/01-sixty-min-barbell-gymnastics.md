# Example 1 — 60-minute plan for a barbell + gymnastics workout (happy path)

**Prompt:** "Turn this into a 60-minute class plan."

**Context:** Workout given —
**21-15-9 for time:** power cleans (95/65 lb · 43/29 kg) + chest-to-bar pull-ups.
Stimulus: short-to-medium sprint-grind; ~6 to 10 min. Standard class, ~14 athletes,
full equipment. Built from
[../../../references/class-planning.md](../../../references/class-planning.md).

---

## Minute-by-minute timeline
| Block | Clock | Coach action |
| --- | --- | --- |
| Pre-class prep | before | Bars/plates/rig counted; whiteboard written (couplet, stimulus, target, scaling map) |
| Arrival / intro | 0–2 | Greet, scan for injuries/absences, frame the day as a short fast couplet |
| Whiteboard brief | 2–5 | Intended stimulus, ~6–10 min target, scaling options for cleans + C2B |
| General warm-up | 5–12 | Row/bike 3 min, then hip/shoulder mobility + scap pull-ups |
| Specific warm-up | 12–24 | Clean drill (pull → extension → receive); kip/pull-up progressions |
| Teaching / skill | 24–32 | Teach C2B (the higher-skill movement): hollow/arch, pull to chest, contact standard |
| Rehearsal / build | 32–42 | 3 power cleans at workout load; short pull-up set at chosen scaling, at pace |
| Setup & brief | 42–45 | Stations set, standards restated, confirm each athlete's scaling |
| Workout | 45–55 | Run the floor; 10-min window covers the cap comfortably |
| Recovery / cooldown | 55–58 | Easy bike/row + lat/forearm stretch |
| Cleanup / notes | 58–60 | Reset bars/rig, log scores and scaling used |

## Per-movement teaching points
- **Power clean (moderate skill):** (1) bar close, fast extension; (2) receive above
  the knee, elbows fast; (3) drop-and-reset over touch-and-go once bar speed drops.
- **Chest-to-bar pull-up (high skill — weighted heavier):** (1) tight hollow/arch
  kip; (2) pull until chest contacts the bar (no-rep if only chin clears); (3) break
  early into small sets to protect grip across 21-15-9.

## Scaling checkpoints
1. During specific warm-up (can the athlete cycle the clean load smoothly?).
2. During rehearsal (does C2B hold up — else regress to pull-ups/jumping/banded).
3. At the setup brief (lock each athlete's load and pull-up scaling on the board).

## Coach positioning
Specific warm-up: front-center to demo. Workout: along the rig-facing side so every
barbell drop zone and the pull-up rig are in view. Step toward any athlete whose
clean midline or kip starts to break down.

## Transitions
Warm-up to teaching: keep bars loaded for the rehearsal. Rehearsal to workout:
athletes stay at their own bar/rig station — no floor reshuffle needed.

## Buffers + late-running contingency
~2 min of slack sits in warm-up/build. **If behind:** trim the build to one clean
single + one short pull-up set; do **not** cut the C2B teaching or the warm-up —
this is a high-skill, grip-intensive piece.

## Coach notes
Watch grip fatigue on the round of 21; cue early breaks. Note who scaled C2B so the
next class plan can progress them. Reset the rig and bars for the following hour.
