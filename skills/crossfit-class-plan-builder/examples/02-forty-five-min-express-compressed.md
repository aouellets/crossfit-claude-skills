# Example 2 — 45-minute express plan (compress blocks proportionally)

**Prompt:** "We only have 45 minutes. Make a class plan for this one."

**Context:** Workout given —
**AMRAP 12:** 200 m run + 12 kettlebell swings (53/35 lb · 24/16 kg) + 8 push-ups.
Stimulus: steady aerobic-glycolytic grind. Lower-skill piece, ~16 athletes. Blocks
are scaled down from the 60-min template in
[../../../references/class-planning.md](../../../references/class-planning.md).

---

## Minute-by-minute timeline
| Block | Clock | Coach action |
| --- | --- | --- |
| Pre-class prep | before | KBs counted, running lane clear, whiteboard written |
| Arrival / intro | 0–1.5 | Greet, scan for injuries, frame as a sustainable aerobic AMRAP |
| Whiteboard brief | 1.5–4 | Stimulus, pacing intent, scaling for swing load + push-ups |
| General warm-up | 4–9 | Easy 200 m jog + dynamic hips/shoulders |
| Specific warm-up | 9–17 | Hip-hinge to swing drill; push-up scaling options |
| Teaching / skill | 17–22 | Brief — KB swing (hinge, not squat; tall plank overhead/Russian standard) |
| Rehearsal / build | 22–27 | 1 short round at pace to set the breathing |
| Setup & brief | 27–29 | Confirm swing load + push-up scaling per athlete |
| Workout | 29–41 | 12-min AMRAP; coach the floor + the run turnaround |
| Recovery / cooldown | 41–43 | Walk + shoulder/hip stretch |
| Cleanup / notes | 43–45 | Reset KBs, log rounds + scaling |

## Per-movement teaching points
- **KB swing (weighted — main lift):** (1) hinge at the hip, flat back; (2) snap
  glutes, arms relaxed; (3) hold standard (overhead vs. eye-level) consistently.
- **Run:** (1) settle a repeatable pace, do not sprint round 1; (2) controlled
  turnaround.
- **Push-up:** (1) rigid plank, full lockout; (2) scale to incline/knees to keep
  reps unbroken.

## Scaling checkpoints
1. Specific warm-up (swing load felt; push-up variant chosen).
2. Setup brief (lock each athlete's load + push-up scaling).

## Coach positioning
Stand where both the KB/push-up floor and the run lane entrance are visible; watch
hinge mechanics and push-up midline as fatigue builds.

## Transitions
Express format keeps everyone at one spot for swings/push-ups; the run is the only
movement off the floor — stagger nothing, it is an AMRAP.

## Buffers + late-running contingency
~1.5 min slack lives in warm-up. **If behind:** shorten the rehearsal to a single
swing set; keep the hinge teaching and warm-up intact. The compression here came
from trimming general warm-up and cooldown, never the safety brief.

## Coach notes
Lower-skill piece, so teaching time is short — spend the saved minutes coaching
pacing on the floor. Note anyone whose hinge degrades under fatigue.
