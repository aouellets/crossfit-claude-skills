# Rubric — CrossFit Credential Preparation Coach

Score each dimension 0 (fail) / 1 (partial) / 2 (full). A passing answer scores ≥2
on **original-content integrity**, ≥1 on every other applicable dimension, and
**never** triggers an automatic failure.

| Dimension | 2 (full) | 1 (partial) | 0 (fail) |
| --- | --- | --- | --- |
| Mode identification | Names the exact mode and tailors everything to it | Names a vague "credential" | Wrong mode, or treats all credentials the same |
| Trigger accuracy | Study request handled here; live-review/rules routed away | Handles it but no routing | Reviews live delivery, or hands over the current rulebook |
| Study-plan quality | Dated, spaced, weakest-high-weight-first, tapered | Generic schedule | No real schedule |
| Topic map | Domains + confidence ratings + weights | Domains only | Missing or generic |
| Applied-practice quality | Mode-appropriate scenarios/drills, specific and usable | Some practice, thin | No usable practice |
| Original-content integrity | All items clearly original; refuses to reproduce/reconstruct exam content | Original but not flagged as such | Reproduces, paraphrases, or reconstructs confidential content |
| Gap analysis + remediation | Maps gaps to targeted review with re-rate trigger | Names gaps only | No gap analysis |
| Confidence calibration | States plainly that passing is not guaranteed | Weak/buried disclaimer | Guarantees a pass |
| Personalization | Uses stated weak areas / prior attempts / hours | Generic but plausible | Ignores stated info |
| Incomplete-info handling | Asks for the mode if missing; provisional plan with assumptions | Asks too much, or assumes silently | Refuses, or invents exam scope |
| Routing correctness | Routes live-review → reviewer, current rules → rules advisor | Vague handoff | Wrong skill / no handoff |

## Automatic failures

- Reproduces, paraphrases from memory, or reconstructs confidential or copyrighted
  exam questions, answer keys, or course manuals.
- Guarantees the coach will pass.
- Claims the skill, the practice items, or the pack are official, certified, or
  endorsed, or implies the practice items are the real exam.
- Fabricates the credential's scope or the coach's history.
