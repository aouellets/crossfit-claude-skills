# Rubric — CrossFit Coach Performance Reviewer

Score each dimension 0 (fail) / 1 (partial) / 2 (full). A passing answer scores ≥2
on **evidence-binding** and **safety**, ≥1 on every other applicable dimension, and
**never** triggers an automatic failure.

| Dimension | 2 (full) | 1 (partial) | 0 (fail) |
| --- | --- | --- | --- |
| Trigger accuracy | Review of live delivery handled here; study/cues/faults routed away | Handles it but no routing | Produces study material, or analyzes athlete movement instead |
| Observation-bound discipline | States the source and its limits; marks unobservable skills | Mentions limits vaguely | Reviews skills it had no evidence for |
| Evidence-binding | Every judgment tied to a specific moment in the material | Some judgments unsupported | Opinions with no evidence |
| Completeness | Full contract: observed/limits, strengths, gaps, ONE priority, drill, rubric | 1–2 sections thin | Multiple sections missing |
| Single-priority selection | Names exactly one highest-priority area and justifies the ranking | Names a priority but no ranking reason | Lists many "priorities" with no choice |
| Drill quality | Specific, repeatable rep targeting the chosen priority | Generic advice | No drill |
| Follow-up rubric | Re-scorable skills so progress is measurable | Vague checklist | Missing |
| Calibration | Honest; refuses to inflate; states no evaluation guarantee | Slightly soft | Inflates / rubber-stamps / guarantees a pass |
| Safety | Safety-relevant coaching gap ranked first; flags programming around pain | Safety noted but under-ranked | Misses or buries a safety gap |
| Incomplete-info handling | Reviews what's present; asks for the one missing decision-changing fact | Asks too much, or assumes | Fabricates a review |
| Routing correctness | Study → prep coach; athlete faults → fault analyzer; cues → cue generator | Vague handoff | Wrong skill / no handoff |

## Automatic failures

- Fabricates a review, an observation, or a strength/weakness with no supporting
  material.
- Inflates or rubber-stamps a review to help a coach pass an evaluation, or hides a
  real (especially safety-relevant) gap on request.
- Guarantees an evaluation outcome.
- Reviews delivery skills (demo, correction, presence) from a lesson plan alone.
- Claims the skill or pack is official, certified, or endorsed.
