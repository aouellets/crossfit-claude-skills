# Example 3 — 6-week aerobic-base run block

**Prompt:** "Give me a 6-week running base block — I gas out on anything long. Keep
some CrossFit. 5 days a week."

**Context:** Intermediate lifter, weak engine, runs maybe once a month, no
injuries. Has access to a track/road and a rower.

---

## Target & tradeoff
Priority: **aerobic base via running** (long-domain engine). Strength is
**maintained, not advanced** for 6 weeks. The build is a **volume ramp**, not an
intensity ramp — running is novel impact, so we ramp conservatively (see
[../../../references/time-domain-framework.md](../../../references/time-domain-framework.md)).

## Block structure
- **Weeks 1–2:** establish easy conversational mileage (Zone 2), low volume.
- **Weeks 3–4:** extend the long run; add one tempo/threshold session.
- **Week 5:** small deload (cut volume ~25%).
- **Week 6:** peak long run + a time-trial retest.

## Primary-quality prescription
Running **3×/week**, mostly easy:
- **2 easy runs:** conversational pace ("could talk in full sentences"), start ~20
  min, add ~10% distance/week. This is the base — most of the volume.
- **1 quality run:** weeks 3–6 only — tempo intervals, e.g. 4×4 min @ "comfortably
  hard" / 2 min easy, building to 5–6×. Work:rest favors aerobic per the
  time-domain framework.
- Cap impact: if shins/knees complain, substitute one run with equal-duration row
  to keep the aerobic dose without the pounding.

## Maintained CrossFit
Keep **2 metcons/week** + maintenance strength (squat/press 1–2×/week, moderate,
no new PRs). Bias metcons away from high-impact lower-body (limit box jumps,
heavy running volume inside metcons) so total impact stays controlled.

## Weekly template (representative, week 4)
- **Mon:** Easy run 30 min → maintenance strength (upper).
- **Tue:** Short non-impact metcon (row/bike + barbell).
- **Wed:** Tempo run intervals.
- **Thu:** Rest / mobility.
- **Fri:** Long easy run → maintenance squat (light).
- **Sat:** Optional easy aerobic (row/bike).

## Interference management
- **Maintenance volume:** 2 metcons/week; engine work dominates, strength holds.
- **Sequencing:** quality run on fresh legs; no heavy squats the day before the
  long run (leg fatigue wrecks running economy and raises injury risk).
- **Shared-fatigue conflicts:** avoid stacking box jumps / heavy lower-body metcons
  with run days — compounded impact is the top overuse risk here.
- **Accepted regression:** strength PRs pause; that's expected.
- **Deload:** week 5 volume cut, plus swap a run for a row at any impact warning.

## Retest & decision rule
Week 6: time-trial (e.g. 5 k run or 20 min for distance). **Advance** to a
threshold-focused block if the base improved; **extend the base** if pacing still
collapses on the long run.

## Risks & stop conditions
Running is novel impact — stop or substitute (row) for shin/knee/foot pain; ramp
volume, not intensity. Watch for doing the easy runs too hard (the classic error
that erases the base stimulus).

## Rationale
Mostly-easy mileage with one weekly quality session and a conservative volume ramp
builds aerobic base while the impact ramp and re-sequencing keep injury risk and
strength loss in check.
