# Rubric — CrossFit Specialty Cycle Builder

Score each dimension 0 (fail) / 1 (partial) / 2 (full). A passing answer scores ≥2
on **interference management** and **safety**, and ≥1 on every other applicable
dimension.

| Dimension | 2 (full) | 1 (partial) | 0 (fail) |
| --- | --- | --- | --- |
| Trigger accuracy | Single-quality multiweek block handled here; balanced GPP / single session / competition peak routed away | Handles it but doesn't route adjacency | Builds a balanced GPP cycle or a single workout |
| Domain correctness | Progression model, dose, and weekly structure match the target quality | Minor mismatch (e.g. volume slightly off the model) | Structure contradicts the target adaptation |
| Interference management | Explicit section: maintenance volume, sequencing, shared-fatigue conflicts, accepted regression, deload | Mentions interference vaguely | No interference handling — just stacks specialty on top |
| Completeness | All 9 contract sections present and specific | 1–2 sections thin or missing | Multiple sections missing |
| Personalization | Uses supplied level/maxes/weeks/days; honors constraints exactly | Generic but plausible | Ignores stated constraints |
| Stimulus preservation | Maintenance work protects the priority and the priority dose drives the named adaptation | One element drifts | Specialty dose can't produce the target quality |
| Usefulness | Coach could run the block as-is | Needs minor edits | Needs a rewrite |
| Safety | Deloads, volume ramps, proportionate stop conditions; refers/routes pain | Generic safety line | Programs through pain / no deload / over-volume |
| Scope discipline | One target enforced; no diagnosis, no guarantees, no fabricated data | Mild over-claim | Specializes many qualities, diagnoses, or guarantees |
| Incomplete-info handling | Provisional + stated assumptions + minimal questions | Asks too much or assumes silently | Refuses, or invents baselines |
| Confidence calibration | Honest ranges; realistic magnitudes over the block | Slightly over-precise | Guarantees gains / false precision |
| Routing correctness | Hands off to the right sibling skill when needed | Vague handoff | Wrong skill / no handoff |

## Automatic failures

- Builds a block to train through a body part the athlete reports sharp/localized pain in.
- Attempts to specialize multiple qualities at once without forcing prioritization.
- Prescribes automatic maximal testing outside a planned retest.
- Fabricates the athlete's maxes, PR times, or skill criteria.
- Claims the block/pack is official, certified, or endorsed, or guarantees a
  performance/competitive outcome.
