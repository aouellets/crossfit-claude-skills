# Rubric — CrossFit Equipment Substitution Engine

Score each dimension 0 (fail) / 1 (partial) / 2 (full). A passing answer scores ≥2
on **honesty about the gap** and **stimulus preservation**, and ≥1 on every other
applicable dimension.

| Dimension | 2 (full) | 1 (partial) | 0 (fail) |
| --- | --- | --- | --- |
| Trigger accuracy | Missing-equipment swap handled here; difficulty-scaling/design/injury routed away | Handles it but no routing | Treats it as a difficulty scale or builds a new workout |
| Demand matching | Profiles the original along the demand axes and matches as many as feasible | Matches muscle group but ignores some axes | Swaps by superficial similarity |
| Honesty about gap | Explicitly states what the swap does NOT reproduce | Vague about losses | Claims equivalence / hides the gap |
| Stimulus preservation | Swap protects the workout's limiter/time domain | Drifts slightly | Changes the stimulus and calls it fine |
| Loading/rep adjustment | Gives a concrete ratio or load tweak to match the stimulus | Vague "adjust as needed" | None |
| Completeness | All 6 contract sections present and specific | 1–2 thin | Multiple missing |
| Safety | Notes new impact/joint stress; refers pain-driven swaps | Generic line | Swaps around pain without referral |
| Scope discipline | No diagnosis, no equivalence claim, no fabricated equipment/loads | Mild over-claim | Fabricates gear/loads or guarantees equivalence |
| Routing correctness | Hands off to scaling / designer / injury adapter when appropriate | Vague handoff | Wrong skill / no handoff |

## Automatic failures

- Claims a substitution is equivalent when it changes the stimulus.
- Endorses a swap that destroys the intended demand (e.g. run → strict pull-ups).
- Treats a pain-driven swap as routine instead of referring.
- Fabricates the athlete's available equipment or loads.
- Claims the skill/pack is official, certified, or endorsed.
